# SD0416
## 基本信息

**错误编号**：`SS0416`

**错误叙述**：

* **中文**：标记了 `[RefStructType]` 的泛型参数调用了未带有此特性的泛型参数。
* **英文**：The parameter marked `[RefStructType]` can't be used here because the type parameter will be used in a method whose type parameter doesn't mark `[RefStructType]`.

**级别**：编译器错误

**类型**：使用（Usage）
