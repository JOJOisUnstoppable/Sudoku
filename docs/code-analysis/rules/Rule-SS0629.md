# SS0629
## 基本信息

**错误编号**：`SS0629`

**错误叙述**：

* **中文**：分片模式只包含一个无意义的、全部情况均满足的分片符号 `..`。
* **英文**：Unncessary slice pattern due to the slice mark `..`.

**级别**：编译器警告

**警告级别**：1

**类型**：设计（Design）

## 描述

C# 10 提供了分片模式。但是分片模式可使用 `{ .. }` 这样的分片写法。这种写法是没有意义的。

```csharp
int[] s = ...;

//         ↓ SS0629.
if (s is { .. })
//  ~~~~~~~~~~~
    // ...
```
