## 基本信息

**错误编号**：`SS9002`

**错误叙述**：

* **中文**：可省略的数组实例化声明。
* **英文**：Redundant array creation statement.

**级别**：编译器信息

**类型**：使用（Usage）

## 描述

数组变量在声明的时候，可以直接书写初始化器，也可以追加 `new` 实例化语句，但是有初始化器的时候，实例化部分是冗余的。

```csharp
//              ↓ SS9002.
int[] arr = new int[] { 1, 2, 3, 4, 5 };
//          ~~~~~~~~~
```
